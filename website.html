Concept & Inspiration

I was hired to make a website for a client who wanted to showcase her photography portfolio online. This was a two-month project from start to finish — I handled the full layout, design, and implementation, and added whatever text and images my client requested.

Design Goals & Challenges

Goals

  • Visually appealing color palette that matched the client's brand
  • Photo galleries displayed via a carousel
  • Responsive design that functions well on both mobile and desktop
  • Simple layout that anyone — regardless of tech experience — can navigate

Challenges

  • Building the carousel feature from scratch
  • Maintaining consistent image sizing across different pages

  • Visual Studio Code
  • HTML5
  • CSS3
  • JavaScript
